# WHOP - Workflow-guided Hospital Outcomes Platform

An Intelligent Decision Support System (IDSS) for ER management, designed to assist players of the simulation game "Friday Night at the ER".

## Overview

WHOP provides real-time decision support for managing emergency room operations, helping users optimize patient flow, resource allocation, and staff management during simulated ER scenarios.

## Features

- Patient triage recommendations
- Resource allocation optimization
- Staff scheduling suggestions
- Real-time performance metrics
- Decision outcome predictions
